# User Guide

## Terminology

- **Design Tab**: The vector editor (Penpot).
- **AI Tab**: The image generator (Fooocus/SDXL).
- **Bridge**: The connection layer that sends images from AI to Design.

## Workflow: From Idea to Post

1. **Generate Assets**: Go to the AI tab. Select "Product Photography" style. Enter prompt: "Blue running shoe". Click Generate.
2. **Select Template**: Go to Templates. Choose "Instagram Product Post".
3. **Assemble**: The template opens in the Design tab. Drag your generated shoe image into the placeholder.
4. **Export**: Click "Export" in the top right to get a PNG/JPG.

## optimization

If you encounter "Out of Memory" errors:

1. Go to **Settings**.
2. Lower **VRAM usage limit** to 6GB.
3. Enable **Tiled VAE**.
